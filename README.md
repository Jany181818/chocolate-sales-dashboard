# 🍫 Chocolate Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Chocolate Sales Dashboard built using Microsoft Power BI. It provides valuable insights into sales performance, profit, shipments, and regional trends. The dashboard helps analyze business performance through interactive charts, KPIs, and filters.

---

## 📖 Short Description
Developed a Power BI dashboard to analyze chocolate sales data and visualize key business metrics such as total sales, profit, monthly performance, product-wise sales, and regional insights. The dashboard enables users to make data-driven decisions using interactive visualizations.

---

## 🛠️ Tech Stack
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel (Dataset)

---

## 📂 Data Source
- Chocolate Sales Dataset (Excel)
- Contains information about:
  - Sales
  - Profit
  - Products
  - Sales Persons
  - Countries/Regions
  - Shipments
  - Monthly Sales
